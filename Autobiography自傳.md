# 自傳

## 一、自我介紹

身為資訊管理系的應屆畢業生，我性格積極主動、具備良好時間規劃與團隊協作能力。大學四年中，我不僅精進系統分析與程式設計基礎，更培養了以使用者需求為導向的思考模式。透過校內專題與外部實習，我習得實戰工程經驗，並具備跨部門溝通與問題解決能力。特別是在後端開發方面，我具備 PHP 程式設計能力，能完成從需求分析、資料庫設計到系統部署的完整流程，達到職場求職水準。

## 二、學習歷程

大學期間，我修習了多門程式設計與資訊管理相關課程，包括：

- **程式語言**：Python、Java、C#、PHP
- **網頁前端/後端**：HTML5、CSS3、JavaScript、PHP (Laravel、CodeIgniter)
- **資料庫管理**：MySQL、SQL Server，熟悉正規化與 ORM 概念
- **系統分析與設計**：UML、需求訪談、流程建模
- **人工智慧與資料探勘**：應用於專題專案中的機器學習模型

在專題實作方面，我曾與團隊共同開發 **AI 牙齒 X 光影像辨識系統**，主要負責資料前處理、特徵擷取與模型訓練。此外，自學並運用 PHP 建置 **線上訂餐系統**，實作會員管理、菜單 CRUD 以及付款流程，並部署於 LAMP 環境，獲得老師高度評價。

## 三、課外活動與實習

- 參與資訊管理學會，擔任活動組成員，規劃迎新活動與技術分享會，提升企劃及執行能力。

學習路上，我持續以專業與責任感要求自己，對 PHP 開發有熱情且具備實作能力，同時熟悉常用的 MVC 架構。面對新技術，我保持好奇，會主動透過文件與實際專案迅速上手。
